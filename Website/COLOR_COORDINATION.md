# Color Coordination Guide

## Logo-Based Color Scheme

The website color scheme is now perfectly coordinated with your Hashimi logo:

### Primary Color: Deep Blue (#003D82)
**Used in logo for:**
- Left part of graphic icon
- "hashimi" brand name text
- "Powered by Life" tagline

**Used throughout website for:**
- Top bar background
- Brand name and tagline in header
- Section headings (h1, h2)
- Primary text elements
- Navigation links
- Footer background (dark variant)
- Hero highlights background

### Secondary Color: Vibrant Orange (#FF6B35)
**Used in logo for:**
- Right part of graphic icon

**Used throughout website for:**
- Primary call-to-action buttons (.btn-primary)
- Button hover states (darker orange #ff5722)
- Navigation link underlines on hover/active
- Slider dot indicators (active state)
- Outline button borders
- Feature icon backgrounds (gradient with blue)
- Hero gradient accents

### Neutral Colors
- **White (#FFFFFF)**: Clean background matching logo background
- **Light Grey (#F5F5F5)**: Alternate section backgrounds
- **Text Grey (#555555)**: Muted text for descriptions

## Color Application Examples

### Buttons
- **Primary buttons**: Orange background (#FF6B35) with white text
- **Hover state**: Darker orange (#ff5722) with enhanced shadow
- **Outline buttons**: Orange border, transparent background, orange text on hover

### Typography
- **Headings**: Deep blue (#003D82) for brand consistency
- **Body text**: Dark grey (#1a1a1a) for readability
- **Muted text**: Medium grey (#555555) for secondary information

### Interactive Elements
- **Navigation links**: Blue text with orange underline on hover/active
- **Hero highlights**: Blue background with blue text
- **Feature cards**: Subtle blue/orange gradient icons

## Dark Mode Support

The color scheme adapts to dark mode while maintaining logo brand colors:
- Deep blue remains primary (slightly adjusted for contrast)
- Orange accents remain vibrant
- Backgrounds switch to dark greys
- Text switches to light colors for readability

## Brand Consistency

All color choices reinforce:
- **Trust & Reliability** (Deep Blue)
- **Energy & Action** (Vibrant Orange)
- **Professionalism** (Clean whites and greys)

This creates a cohesive visual identity that matches your logo perfectly.
